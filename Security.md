# Security and Abuse Contact

We take security and responsible use seriously. If you believe this tool is being used maliciously, wish to report abuse, or have identified a security vulnerability, please follow our responsible disclosure guidelines.

## Reporting an Issue
To report third-party abuse or security inquiries, please use one of the following methods:

* **Repository Issues:** Open a detailed issue in this repository. If the report contains sensitive vulnerability data, please request a secure communication channel before posting technical details.

All reports will be reviewed promptly, and action will be taken against unauthorized or malicious use where applicable.
